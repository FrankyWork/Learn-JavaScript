Чему будут равны переменные a, b, c и d в примере ниже?

`let a = 1, b = 1;`

`let c = ++a; // ?` 

`let d = b++; // ?`

Ответ:

`let a = 1, b = 1;` - a = 1 , b = 1

Префикс новое значение
`let c = ++a; // ?` - c = 2

Постфикс старое
`let d = b++; // ?` - d = 1

Если мы выведим `let a = 1, b = 1;` сейчас то они будут равны `let a = 2, b = 2;`
